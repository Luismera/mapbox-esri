# MAPBOX ESRI

Este proyecto se creo con el fin de integrar ESRI con un plugin de mapbox

### Instalación

requiere [Node.js](https://nodejs.org/) y [Angular cli](https://cli.angular.io/).

Instale las dependencias

```sh
$ cd mapbox-esri
$ npm install
```

Despliegue la aplicacion...

```sh
$ ng serve -o
```
